# Project Overview

This project focuses on analyzing Facebook & Instagram ad campaign performance using Power BI.

The dashboard provides actionable insights into:

Campaign reach & impressions
User engagement behavior
Conversion performance
Budget utilization

It helps marketing teams optimize ad spend and improve ROI(Return On Investment) through data-driven decisions.

# Business Objective

The goal of this dashboard is to:

- Identify the most effective platform (Facebook vs Instagram)
- Track campaign ROI and conversion funnel
- Analyze audience engagement patterns
- Optimize budget allocation across campaigns

# Key KPIs
- Impressions
- Clicks
- Shares
- Comments
- Purchases (Conversions)
- Engagements (Clicks + Shares + Comments)
- CTR (Click Through Rate)
- Engagement Rate
- Conversion Rate
- Purchase Rate
- Total Budget
- Avg Budget per Campaign

# Dashboard Features
 1. Gender-Based Performance (Donut Chart)
    Compare campaign metrics across gender
    Identify high-performing audience segments
2. Age Group Analysis (Bar Chart)
   Analyze engagement across age categories
   Discover most responsive audience groups
3. Geographic Insights (Map Visualization)
   Country-wise performance breakdown
   Identify high-performing regions
4. Monthly Trends (Calendar Heatmap)
   Detect seasonal trends
   Identify peak campaign months
5. Weekly Trends by Ad Type (Stacked Column Chart)
   Compare ad formats performance over time
   Track weekly campaign trends
6. Hourly Activity Analysis (Area Chart)
   Understand user activity patterns throughout the day
   Optimize ad timing strategy
7. Ad Type Performance Matrix
   Compare performance across:
   Ad Types
Platforms (Facebook vs Instagram)

# Key Insights 
- Identify which platform drives higher conversions
- Discover peak hours for maximum engagement
- Understand which age group & gender contributes most
- Optimize campaigns based on high-performing ad types
